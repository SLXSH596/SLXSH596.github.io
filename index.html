About Me

I am a second year Computer Games Development student at Staffordshire University. I have always had a passion for game development, and started programming in middle school. I am always looking to learn new skills and improve my existing ones. I am currently learning blender and developing a game in UE5 alongside my studies.

Projects

Rod & Ripple

Rod & Ripple is a fishing game developed in Unreal Engine 5. This game is the result of my Junior Collaborative Games Development Module. On this project, I worked on:

  • Input handling
  • Boat mechanics
    • Boat movement
    • Boat camera movement
    • Boat health
  • HUD and UI design, implementation and functionality
    • Boat HUD
    • Casting HUD
    • Shop UI
    • Menus
  • Hub area mechanics
  • Bait system

This project furthered my knowledge of Blueprinting and UI development in Unreal Engine 5. It also furthered my knowledge about team development, and of the use of GitHub for version control in large groups and collaboration.

Railway System Tool

Railway System Tool is a tool developed by myself in Unreal Engine 5.4 for quickly and easily creating and managing railway systems.

This project has taught me about the use of splines, C++ and Editor Utility Widgets in Unreal Engine 5. I intend to update and continue development of this tool, adding more features and functionality to it. I will release this tool on the FAB Marketplace when it is ready.

Hexwave

Hexwave is a first person, arcade style shooter developed by myself in Unreal Engine 5. The goal of the game is to survive increasingly difficult waves of enemies. This project taught me about the use of interfaces, behaviour trees, animation and more.

Wasted Knight

Wasted Knight is a first person roguelike melee game developed by my team of 4 developers. From this, I learned about the whole games production process, and used agile development within a small team to produce an impressive result in 9 weeks.
